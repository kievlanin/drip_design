Метод Ньютона-Рафсона для гідравлічного розрахунку латералі



Цей алгоритм використовується для швидкого знаходження тиску на кінці латералі (Hend​), при якому тиск на вході збігається з фактичним тиском у сабмейні (Htarget​).

Математична база



Ми шукаємо корінь функції помилки:

f(Hend​)=Hcalc\_start​(Hend​)−Htarget​=0



Згідно з методом Ньютона, наступне наближення Hend​ розраховується як:

Hnext​=Hcurr​−f′(Hcurr​)f(Hcurr​)​



Де f′(Hcurr​) — це похідна функції тиску на вході по тиску на виході. Оскільки аналітично її вирахувати складно через дискретність емітерів, використовується чисельне диференціювання.

Алгоритм реалізації:



&#x20;   Початкове наближення (H0​):



&#x20;       Припустимо, що втрати складають 10-15%.



&#x20;       Hcurr​=Htarget​×0.9



&#x20;   Ітераційний цикл (поки ∣f(Hcurr​)∣>Tolerance):



&#x20;   Крок А: Розрахунок значення функції



&#x20;       Викликати backwards\_step\_method(H\_curr) → отримати Hstart1​.



&#x20;       f(Hcurr​)=Hstart1​−Htarget​.



&#x20;   Крок Б: Чисельне знаходження похідної (Метод скінченних різниць)



&#x20;       Додати мале зміщення: ΔH=0.001 м.



&#x20;       Викликати backwards\_step\_method(H\_curr + \\Delta H) → отримати Hstart2​.



&#x20;       Розрахувати похідну:

&#x20;       f′(Hcurr​)=ΔHHstart2​−Hstart1​​



&#x20;   Крок В: Оновлення значення



&#x20;       Hnext​=Hcurr​−f′(Hcurr​)f(Hcurr​)​



&#x20;       Hcurr​=Hnext​



&#x20;   Критерій зупинки:



&#x20;       Якщо ∣Hnext​−Hcurr​∣<Tolerance або кількість ітерацій > 50.



Переваги для нашого проекту:



&#x20;   Швидкість: Newton-Raphson має квадратичну збіжність. Для вашого CAD-інтерфейсу це означає миттєвий перерахунок гідравліки при зміні положення латералі на мапі.



&#x20;   Точність: Метод дозволяє знайти тиск з точністю до міліметра водного стовпа.



Можливі ризики:



&#x20;   Від'ємні значення: Якщо Hnext​ стає менше 0 (можливо при дуже великих втратах), додайте перевірку: if H\_next < 0: H\_next = H\_curr / 2.



&#x20;   Нульова похідна: У гідравліці це малоймовірно, оскільки система монотонна (чим вище тиск в кінці, тим вище він на початку).

def solve\_lateral\_newton(H\_target, tolerance=0.001):

&#x20;   h\_curr = H\_target \* 0.85  # Початкова точка

&#x20;

&#x20;   for \_ in range(20):

&#x20;       # f(h)

&#x20;       h\_start1, \_ = backwards\_step\_method(h\_curr)

&#x20;       error = h\_start1 - H\_target

&#x20;

&#x20;       if abs(error) < tolerance:

&#x20;           return h\_curr, h\_start1

&#x20;

&#x20;       # Чисельна похідна f'(h)

&#x20;       delta = 0.001

&#x20;       h\_start2, \_ = backwards\_step\_method(h\_curr + delta)

&#x20;       derivative = (h\_start2 - h\_start1) / delta

&#x20;

&#x20;       if derivative == 0: break

&#x20;

&#x20;       h\_curr = h\_curr - (error / derivative)

&#x20;

&#x20;   return h\_curr, h\_start1
